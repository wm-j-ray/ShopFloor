## Tension Level Key

1 = minimal — breathing room, character texture, world detail
2 = low — mild pressure, situational friction
3 = moderate — active conflict, stakes present
4 = high — escalating pressure, decision under duress
5 = peak — maximum stakes, irreversible action

## Tempo Key

fast     = high event density, short scenes, urgent prose
moderate = balanced event and interiority
slow     = reflection, atmosphere, aftermath, character texture

## Function Type Key

exposition     = world or character information delivered
escalation     = conflict intensified
reversal       = expectation or situation inverted
revelation     = withheld information released
confrontation  = direct clash between characters or forces
transition     = movement between states or locations
resolution     = conflict closed or partially closed
