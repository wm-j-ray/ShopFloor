# Plot Thread Tracker

> **Purpose:** Follows a single narrative thread — a question, pursuit, or unresolved force — across multiple scenes and chapters regardless of which character carries it.
>
> **Instance naming:** When creating an instance, save as `[ID]_Name.md`, set `profile_version` and `date_instantiated`.

## Thread Status Key

dormant    = thread exists but is not active in this scene
active     = thread is being pursued or pressured
blocked    = forward movement prevented
advanced   = thread moves closer to resolution
resolved   = thread closed
abandoned  = thread dropped intentionally or by accident
